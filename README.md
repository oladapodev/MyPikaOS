# 🎨 My PikaOS Demon Slayer Setup

This repository contains my personal PikaOS configuration, including GNOME themes, shell enhancements, and boot manager customizations.

## 📸 Features
- **OS:** PikaOS (GNOME)
- **GTK Theme:** WhiteSur-Dark (MacOS Traffic Lights)
- **Shell Theme:** Orchis-Dark-Compact
- **Icons:** Slot-Beauty-Dark-Icons
- **Cursor:** FernBLZ (Animated)
- **Shell:** Bash + ble.sh (Highlighting/Autosuggestions) + Starship + eza + zoxide + fzf
- **Bootloader:** rEFInd Demon Slayer Theme (Tanjiro/Zenitsu)

## 🚀 One-Click Install

Clone this repo and run the install script:

```bash
git clone https://github.com/YOUR_USERNAME/my-pikaos-dotfiles.git
cd my-pikaos-dotfiles
chmod +x install.sh
./install.sh
```

## 🔑 Post-Installation Steps

### 1. rEFInd Bootloader
To enable the Demon Slayer boot screen, run:
```bash
echo 'include themes/rEFInd-tanjiro/theme.conf' | sudo tee -a /boot/efi/EFI/refind/refind.conf
```

### 2. Login Screen (GDM)
1. Run `gdm-settings`.
2. Set **Shell**, **GTK**, and **Icons** to match your desktop themes.
3. Set the background to `assets/wallpaper.png`.
4. Click **Apply**.

### 3. Terminal Font
Open your terminal settings and set the font to **"JetBrainsMono Nerd Font"** so the icons show up correctly.
